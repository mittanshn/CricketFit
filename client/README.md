# CricketFit

CricketFit is a full-stack cricket training and fitness tracking platform designed to help players organize practice sessions, log workouts, monitor progress, and improve performance over time.

## Features

- Cricket training planner
- Batting and bowling workout logs
- Fitness workout tracking
- Progress analytics dashboard
- AI-generated training recommendations
- Mobile responsive UI

## Tech Stack

### Frontend

- React
- TypeScript
- Vite
- Tailwind CSS

### Backend

- Node.js
- Express

### Database

- PostgreSQL
