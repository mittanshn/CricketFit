# CricketFit Data Analytics
This folder has the data analytics and recommendation logic for CricketFit.

# Features
- Reads cricket training logs from CSV
- Groups practice data by week
- Calculates batting, bowling, and fitness minutes
- Tracks average fatigue and performance
- Gives role based weekly recommendations
- Creates weekly summary output for dashboard use

# Current Roles Supported
- Batsman
- Bowler
- Batting All-Rounder
- Bowling All-Rounder

# Files
- data.csv: sample training log data
- analysis.py: analytics and recommendation logic
